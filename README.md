# Food Kiosk Backend

This project is the backend API for a self-service food ordering kiosk system.  
It is built with Laravel and provides endpoints for managing menu items, categories, and customer orders.

The API is consumed by a React-based kiosk interface that allows customers to browse the menu and place orders through a touchscreen-style interface.

## Features

- REST API for kiosk application
- Menu and category management
- Order creation and processing
- Database integration
- Authentication for administration 
- Manage users

## Tech Stack

- Laravel
- PHP
- MySQL
- REST API

## Purpose

This project demonstrates building a backend API with Laravel to support a real-world ordering system.
